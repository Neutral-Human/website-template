# Website Template
これはneutral-human.netで使用されているWebページのソースを一部リライトして公開したものです。<br>
不具合や設計ミス等あると思われますのでPull RequestやIssues等大歓迎です。
## はじめかた(超短縮版)
プロジェクトルートで以下のコマンドをNode.jsがインストールされた環境で実行すると開発サーバーが起動します。
```
npm install
npm run dev
```
## その他
このテンプレートを使用する場合は、特に記載等はいりませんがもし可能であればこのテンプレートを使用したことを明記してくださるとありがたいです。
